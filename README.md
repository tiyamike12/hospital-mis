# Hospital Management System
This is a simple CRUD web system for managing common hospital data.
Built in Java and Spring Boot 2.6

# Getting Started
### Dependencies
* Spring Web
* Spring Security
* Thymeleaf
* Spring Data Jpa
* PostgreSQL
* Flyway
* Java Validator
* Java Mail Sender
* Lombok
